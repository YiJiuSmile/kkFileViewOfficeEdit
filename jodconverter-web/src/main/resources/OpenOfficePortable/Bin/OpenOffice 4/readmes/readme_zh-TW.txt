是否任何使用者真的都可免費使用 OpenOffice？ 
----------------------------------------------------------------------

每個人皆可免費使用 OpenOffice。您可以取得 OpenOffice 軟體，安裝在不限數目的電腦上，進行多用途的使用 (包括商用、政府、公共事業管理及教育之用)。如需進一步的詳細資訊，請參閱 OpenOffice 隨附的授權文字或 http://www.openoffice.org/license.html

為何所有使用者皆可免費使用 OpenOffice？
----------------------------------------------------------------------

您今天之所以可免費使用 OpenOffice，是因為個人參與者及企業贊助者的設計、開發、測試、翻譯、記載、支援、行銷，以及透過許多其他方式的協助，OpenOffice 才有今天的成就，成為世界領先的開放原始碼辦公室軟體。

如果您欣賞並感謝他們的貢獻，並希望 Apache OpenOffice 未來能夠繼續運作，請考慮貢獻這個專案 - 請看此網頁 http://openoffice.apache.org/get-involved.html 獲得更多有關貢獻細節和捐款細節網頁 http://www.apache.org/foundation/contributing.html 每個人都能做出貢獻。

安裝注意事項
----------------------------------------------------------------------

OpenOffice 需要最新的JAVA版本來支援完整的功能。JAVA可以從此下載： http://java.com 。

系統需求
----------------------------------------------------------------------

* Microsoft Windows XP 、 Vista 、 Windows 7 或是 Windows 8
* Pentium III 或是更新的處理器
* 256 MB 記憶體 (建議使用 512 MB 記憶體)
* 最大 1.5 GB 的可用硬碟空間
* 1024x768 解析度 (建議使用更高解析度)，至少 256 色

請注意，安裝程序需要管理員權限。

執行安裝程式時使用以下指令行開關，可強制或不讓 OpenOffice 註冊為 Microsoft Office 格式的預設應用程式：

* /msoreg=1 會強制註冊 OpenOffice 為 Microsoft Office 格式的預設應用程式。
* /msoreg=0 則不會註冊 OpenOffice 為 Microsoft Office 格式的預設應用程式。

如果使用setup /a 執行管理安裝，您必須確定系統上已安裝檔案 msvcr100.dll。管理安裝之後，OpenOffice 需要此檔案才能啟動。您可以從 http://www.microsoft.com/en-us/download/details.aspx?id=5555取得檔案。

請注意，安裝程序需要管理員權限。

請確定系統上的暫存目錄有足夠的可用記憶體，並已授與讀取、寫入和執行存取權。請關閉所有其他程式，再開始安裝。

程式啟動期間發生問題
----------------------------------------------------------------------

無法啟動 OpenOffice (例如應用程式停滯) 以及螢幕顯示發生問題，通常是由於圖形卡驅動程式所致。若發生這些問題，請更新您的圖形卡驅動程式，或嘗試使用作業系統所提供的圖形驅動程式。停用「工具」-「選項」-「OpenOffice」-「檢視」-「3D 檢視」下的「採用 OpenGL」選項，通常可解決無法顯示 3D 物件的問題。

----------------------------------------------------------------------
Windows 中的 ALPS/Synaptics 筆記型電腦觸控板
----------------------------------------------------------------------

由於 Windows 驅動程式發生問題，您無法在 ALPS/Synaptics 觸控板上滑動手指時捲動 OpenOffice 文件。

若要啟用觸控板捲動功能，請將以下行加入「C:\Program Files\Synaptics\SynTP\SynTPEnh.ini」配置檔案，然後重新啟動電腦：

[OpenOffice]

FC = "SALFRAME"

SF = 0x10000000

SF |= 0x00004000

配置檔案的位置會隨 Windows 版本而異。

----------------------------------------------------------------------
捷徑鍵
----------------------------------------------------------------------

OpenOffice 中只能使用作業系統未使用的捷徑鍵 (按鍵組合)。如果 OpenOffice 中的按鍵組合執行方式與 OpenOffice 說明中的說明不相符，請檢查作業系統是否已使用該捷徑。若要修正此衝突，您可以變更作業系統所指定的按鍵。或是變更 OpenOffice 中幾乎所有的按鍵指定。如需有關此主題的更多資訊，請參閱 OpenOffice 說明或作業系統的說明文件。

----------------------------------------------------------------------
從 OpenOffice 以電子郵件傳送文件時發生問題
----------------------------------------------------------------------

透過「檔案」-「傳送」-「作為電子郵件的文件」或「文件當作 PDF 附件」傳送文件時，可能會發生問題 (程式故障或停滯)。這是由於在某些檔案版本中導致問題的 Windows 系統檔案「Mapi」(訊息應用程式發展介面) 所致。不過，此問題無法縮小範圍到特定的版本編號。如需更多資訊，請造訪 http://www.microsoft.com 並在 Microsoft 知識庫中搜尋「mapi dll」。
